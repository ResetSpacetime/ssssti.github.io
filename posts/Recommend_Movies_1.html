【逢十】系列第一篇

这个系列用来小结一些事情，目前可能会多为书影方面，顺带推荐。本篇为第一篇，2015年的前10部观影小结，从1月1日至1月20日，追评时间4月15日。

加了一项个人建议，仅供参考。毕竟看电影还是比较个人化的选择。个人建议分为四档：建议不看、爱看不看、推荐观看、不得不看，“建议不看”出现机会应该会比较小，毕竟看的片子都是自己反复选过的。

相关豆列：【2015观影】

1.寻找隐世快乐（豆瓣7.9 / IMDB7.2）

  • 个人建议：

    • 推荐观看

  • 当时短评：

    • 当下即是无限。语言陷阱。OST大赞。

  • 现在追评：

    • 虽然最后的答案还是可以归结为“冲破束缚，与当下的自己和解”，但画面和OST都太美，加之对束缚的深入思考，论及轮回、灵魂，以出租车的视角遍历人生百态。

2.马达加斯加的企鹅（豆瓣7.4 / IMDB6.8）

  • 个人建议：

    • 爱看不看

  • 当时短评：

    • 看脸的世界。萌无止境。

  • 现在追评：

    • 似乎，记住的就只剩下那只蠢萌的熊和赫尔佐格的配音了。

3.真探（迷你剧）（豆瓣9.2 / IMDB9.3）

  • 个人建议：

    • 不得不看

  • 当时短评：

    • 结局变成了基老伴……神秘主义，存在主义恐怖，怪谈小说，罗伯特·钱伯斯，洛夫克拉夫特，克鲁苏神话……

  • 现在追评：

    • 哈里森和麦康纳两人飙戏实在过瘾，编剧非常有耐心地在前三集进行铺陈，一点点引出恶、引出神秘议题，不断的倒叙和主角的性格转变并行推进，引人入胜，步步惊心。
    • 近一年来一直对死亡和悲观好奇，《真探》给出了一种解答，“神秘主义”和哲学思绪在这部片子中体现甚多。看了很多资料，从美国的南方文学渊源到洛夫克拉夫特对后世恐怖题材的影响，最后又涉猎到一些国外的奇幻主题。默许未知、默认有比人类深厚而隐秘的力量存在，现在看来更是一种新的视角。人类曾对世界有无数种解读并传承许久，对人类诞生的感知和灾难过后的感恩，承认自己的弱小并对死亡心存敬重，这是现代社会正在吞噬并遗弃的。

4.辉夜姬物语（豆瓣8.2 / IMDB8.1）

  • 个人建议：

    • 爱看不看

  • 当时短评：

    • 不如离去

  • 现在追评：

    • 朴朴实实地讲完一个故事，关于爱、自我、人生，以及父母皆祸害。每个人都曾对世界愤怒、失望，但在离去之前却都是一致的留恋，缅怀所有过往。

5.陆上行舟（豆瓣8.8 / IMDB8.1）

  • 个人建议：

    • 不得不看

  • 当时短评：

    • 伟大被未知驾驭着。那些没有解释且略显神秘的镜头太让人着迷。那个厨子最有趣，喝着酒抱着女人时而热烈时而宁静，能与一切融为一体。

  • 现在追评：

    • 看了《在世界尽头相遇》之后就变成赫尔佐格的脑残粉，搞了全集，打算今年内慢慢补完，到现在看完了四五部。他的电影特色在于存在未知，即使像《陆上行舟》这样的剧情片，也一样充满未知。拍摄地选在雨林里，有语言不通的土著人出境，帮着搬运船是因为觉得有神降临……把拍摄中的未知和思考都一股脑放在作品里。因此他的电影不是静态的。
    • 另外，主角只是想做一件事而已，总把它理解为疯子和偏执，是这世界的偏见。

6.心花路放（豆瓣7.0 / IMDB6.5）

  • 个人建议：

    • 爱看不看

  • 当时短评：

    • 过日子，也不只是为了过日子吧？男人啊，总是觉得自己是对的，讽刺得好。郝义那种想车震就车震想接吻就接吻是真生活。

  • 现在追评：

    • 我想，宁浩是想提醒现实生活中的人们珍惜眼前吧，但故事过于男性视角了。对于分手我没有经验，但对于守护过往，我很赞成。

7.彗星来的那一夜（豆瓣8.3 / IMDB7.1）

  • 个人建议：

    • 推荐观看

  • 当时短评：

    • Don’t Panic ^_^ 导演你确定不再拍部续集吗？

  • 现在追评：

    • 轻微烧脑。未来是当下一切可能的叠加，没有可能或不可能，只有发生或没发生。与《这个男人来自地球相比》，缺失了情感和人性。

8.爆裂鼓手（豆瓣8.6 / IMDB8.6）

  • 个人建议：

    • 爱看不看

  • 当时短评：

    • 嘿！这爱情片真不错，秒杀13年阿黛尔的生活了！

  • 现在追评：

    • 节奏、配乐、主角角力都有看点，但整体不太容易解读。“弄死老师”是本片SM属性的极致。不过有趣的是：这般努力的结果，也仅仅是种流行乐罢了，严谨加努力，最后成为了一件优秀的工具。

9.小亚细亚往事（豆瓣7.4 / IMDB7.8）

  • 个人建议：

    • 不得不看

  • 当时短评：

    • 神话与真相。死亡的控诉。突然转折的结果。锡兰这颗苹果像极了赫尔佐格拍的那只企鹅。

  • 现在追评：

    • 反映普遍人性的片子，朴实却厚重。导演只拍下了真相的50%，另一半不可见的真相需要通过思考得到。困扰自己的问题，从与别人的对话中找到答案。大路向天边延展，颜色晦暗，车灯划一道明亮的线，近至眼前。有一搭没一搭地对话，却句句都在引出问题。
    • 现在依然印象深刻的，是那颗自己从果树上落下，随着水流载浮载沉，最后搁浅的苹果。世人所能观察到的，往往就只是搁浅了的事实。时间流逝，大风吹过，真相不易寻觅，于是有了种种解读与猜测，你可以知晓它从哪条小溪漂流而下，但可能永远无法知晓它从哪棵果树上落下。透过这个长镜头，可以映射全片。

10.吸血鬼生活（豆瓣7.9 / IMDB7.6）

  • 个人建议：

    • 推荐观看

  • 当时短评：

    • 蛤蛤蛤。请找到你自己的穿衣风格。

  • 现在追评：

    • 吸血鬼的幕后生活。导演和编剧在戏里玩得不亦乐乎，集古今中外吸血鬼合集。本是恐怖片主角，却被注入了逗逼属性，莫名地显出一种人性的真实。
