IPFS 安装（Windows 系统）

2022-02-21 | Hexo | 阅读 | 635 字 | 2 分钟
  • 注1：本文仅记载了 Windows 系统安装的过程
  • 注2：小白折腾，专家见笑

IPFS 介绍：星际文件系统- 维基百科、IPFS Documentation、IPFS 中文开发文档、IPFS 社区

下载安装 IPFS Desktop

  1. 打开 GitHub - ipfs/ipfs-desktop，下载 Windows 版本；
  2. 打开 .exe 可执行文件进行安装；
  3. 打开 IPFS Desktop，注意：可能需要全局翻墙后，才能顺利运行。

下载 go-ipfs 终端

  1. 打开 IPFS Dist - go-ipfs，下载 Windows 终端压缩包；
  2. 在本地解压缩；
  3. 将解压后的文件夹，整体转移至一个稳定的、不会经常修改的路经下，比如我转存到了 User\AppData\local 文件夹下，如果不常变动，存至本地文档文件夹 User\Documents 下也可以；
  4. 进入文件夹，进入 go-ipfs 文件夹，记录 go-ipfs 文件夹的路径（在地址栏复制），如：C:\Users\AppData\Local\go-ipfs_v0.12.0_windows-amd64\go-ipfs。

系统环境变量设置

  1. Windows - 开始 - 设置，搜索“环境变量”，即“编辑系统环境变量”，点击打开；
  2. 在弹出的“系统属性”对话框右下角，点击“环境变量”；
  3. 在弹出的“环境变量”对话框内，在“系统变量”内，点击“新建”，注意：是在“系统变量”内新建；
  4. “变量名”填写 path，“变量值”填写前一步第 4 项获得的 go-ipfs 文件夹的路径，即 C:\Users\AppData\Local\go-ipfs_v0.12.0_windows-amd64\go-ipfs；
  5. 逐层点击“确定”，将环境变量生效。

测试 ipfs 生效

使用其他命令行软件，输入 ipfs help 回车。不报错，且可正常显示 “USAGE”、”SYNOPSIS”、”OPTIONS” 等选项信息，即已配置生效。

其他备忘

pwd 命令

逐层使用 cd ls 命令（此处最好使用 Windows 自带的 PowerShell，打开方式为 Win+R，输入”powershell”），进入到 go-ipfs 所在路径后，在命令行输入 pwd，即可显示目前所处的路经，可在命令行内复制备用。

在配置环境变量前，测试 ipfs 是否可用

逐层使用 cd ls 命令，进入到 go-ipfs 所在路径后，在命令行（使用 PowerShell，打开方式见前）输入 .\ipfs.exe --version，回车，正常显示版本信息，如 ipfs version 0.8.0，即可正常使用。
