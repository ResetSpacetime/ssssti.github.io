2020 阅读（上）

2020-08-04 | 阅读 | 1.5k 字 | 5 分钟

Jan

《遗失的黄金国度：南美神话》
荷兰时代生活图书公司 / 樊英译 / 中国青年出版社 / 2006
1.3 人类起源真是个好玩的问题。止不住的伤痛从文字里冒出来，就像集中营幸存者们的无声呼喊。

《第七人》
[英国] 约翰·伯格 、 [瑞士] 让·摩尔 / 刘张铂泷译 / 中国美术学院出版社 / 2019
1.5 这是所有移民的底色，X 漂的人们底色也是如此，读至过去当下未来的表述时，才将内容与自己的经验联系了起来。是的，这只是一种更开放、更易迷失的囚禁。对寄居之所，周身事物的无感和无关联，没有归属和相互融入的动力，因为一切都将被交换被计算。都是自我选择的抽离。

《幸运者》
[英国] 约翰·伯格 、 [瑞士] 让·摩尔 / 黄月译 / 中国美术学院出版社 / 2019
2.6 啊，伯格的书，总是有像做了梦一样的阅读体验。一个特别的人与自我，与外界，与周遭，与不断出现的矛盾相背离。如何自处、调和，已成为现代人的永恒问题。

Feb

《猪的土地》
[英国] 约翰·伯格 / 周成林译 / 理想国 广西师范大学出版社 / 2019
2.26 真的还有机会放低吗，再次从土地中生长出来，再瞧瞧自己，正是因为没有了这样的机会，才令人懊恼不已。那是“崭新”的世界，一切都分离又同时一体，缺少对立与争斗的必要。古旧的言辞，原生的鲜活，无尽的遗忘似乎都被他们守护了。灵光铺天盖地，就像泥土与空气，无从触达，唯死亡与它们近似。

《到婚礼去》
[英国] 约翰·伯格 / 郑远涛译 / 广西师范大学出版社 / 2019
2.29 翻译不够口语。应做有声版。中介物性质不明确。当下的词语与其远方的归属。

Mar

《林中的陌生人》
[美国] 迈克·芬克尔 / 匪石译 / 人民文学出版社 / 2019
3.1 预设主题“孤独”太明显。他身份未知时的内容，更有趣些。本就是“无缘由”的逃离。

《惧恨拉斯维加斯》
[美国] 亨特·S·汤普森 / 经雷译 / 理想国 / 广西师范大学出版社 / 2016
3.2 十年前看就好了

《巫医、动物与我》
[澳大利亚] 赫伯特·雷布汉 / 林小绿译 / 广西师范大学出版社 / 2019
3.14 你不了解你自己，而且你害怕知道这个真相

《本雅明档案》
德国瓦尔特·本雅明档案馆编 / 李士勋译 / 北京师范大学出版社 / 2019
3.22 档案化。用于出示的“垃圾”。难以表达的重视。收集者的导师。

《创新者的窘境》
[美国] 克莱顿·克里斯坦森 / 胡建桥译 / 中信出版社 / 2014
3.22 倒是有一点启发，不过重要的是，都可以整合到生态学之下。技术演化上，也没宏观视角，只是数据和讯息的整理。

《1929 年大崩盘》
[美国] 约翰·肯尼斯·加尔布雷思 / 上海财经大学出版社 / 2006
3.29 中产占据消费的绝大多数，算是比较认可的首要原因。但很难从个人角度分析出真切的问题来，或者说，已完全无从归因。

Apr

《迎向灵光消逝的年代》
[德国] 瓦尔特·本雅明 / 许绮玲、林志明译 / 广西师范大学出版社 / 2004
4.5 倒是专心与散心，让我受了些新的启发。最近对“仪式”的一些想法，居然又被本雅明抢先讲了。

《我的天才女友》那不勒斯四部曲之一
[意大利] 埃莱娜·费兰特 / 陈英译 / 人民文学出版社 / 2017
4.15 好奇作者将如何将故事深入至隐蔽的世界中。但这种展开我是挺信服的。

May

《我的经验与教训》
[美国] 苏世民 / 赵灿译 / 中信出版社 / 2020
5.3 见识一下专业服务行业

《锥子》
[韩国] 崔圭硕 / 苏梦芸译 / 贵州人民出版社 / 2020
5.16 像看了一部电影，而图文的陈述比动态影像更能反映出不被重视的背景讯息，或者说这作品补全了很多背景讯息。把读过的劳工抗争历史都投在了现实中，抗争、牺牲与彼此护佑，充满细节。东亚权力来自资源稀缺，得失分明，世袭资质在法的庇护下占据过多主动，要打破的是社会的习惯、执念和怯懦，本就没什么是天生的。

Jun

《跨越边界的社区：北京“浙江村”的生活史》
项飙 / 生活•读书•新知三联书店 / 2018
6.8 温州人真的勇敢。“顶”、“抬”、“套”都是创造出来的工具，时间长了其实也已变成本地文化的一部分，只是没有什么名正言顺。在北京喜欢叫老师，不知与“老司”是否也有关联。这种建构性实属难得，哪怕没有被正名，也已在文化和生活中留下了印记，只是没人深挖。权力与国家的一步步入侵，然后弃之不理，本地与外乡人的政策割裂，自由发展空间不足。北漂也应拥有历史。

《古都》
朱天心 / 九州出版社 / 2018
6.13 更喜欢前几篇，而非同名的《古都》，有点儿读约翰伯格的感觉，但密度似乎高些，喜欢都讲出来，不遮掩，因为都看在自己眼里，不必猜测。所以文中有种炙热。
