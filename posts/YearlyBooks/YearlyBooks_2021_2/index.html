2021 阅读（下）

2021-12-23 | 阅读 | 2.1k 字 | 7 分钟

Jul

《关键路径》
[美国] 理查德·巴克敏斯特·富勒 / 李林、张雪杉译 / 广西师范大学出版社 / 2020
7.3 很多近乎直觉的观察和推演，乐观且自信。前半本非常爽，后半本很受制于当时的技术，不太严谨。现代人真的逊爆了。从教育开始，从个人开始，从自由地思考和行动开始。自然确实会安排一切。对“进动”的阐释简直伟大。

《浮木》
杨本芬 / 北京联合出版公司 / 2021
还是会受到巨大的冲击，平淡的言语，每个词都非常结实，有无法无视的力量

Aug

《意大利风味》
[瑞士] 瓦莱丽·洛萨著 / 丁晓译 / 山东教育出版社 / 2017
8.1 细腻的，忧伤的，刻在了骨子里的，无法背叛的印记。

《伊索寓言》
[古希腊] 伊索著 [意大利] 西蒙·雷亚绘 / 马骊译 / 山东教育出版社 / 2017
8.1 画得太好了，有点抽象，但直抵目标。寓言，与其说是故事或讽刺，不如说成是一种表达欲望的突然爆发与捅破，直白的刺激或质问，且不要求反馈。

《鸟儿》
[荷兰] 泰德·范·利斯豪特文 [荷兰] 路德维希·福贝达绘 / 王奕瑶译 / 山东教育出版社 / 2018
8.1 明线与暗线，信息与语言，诗意的丧失，意志对可能性的驱逐。

《万事通先生》
[丹麦] 本特·奥莱森·尼斯特罗姆 / 马文姬 / 山东教育出版社 / 2017
8.1 是要反复反复反复翻的那种绘本，内含巨量的内容与巧思，以及交流的渴望。

《写给孩子们的故事》
[美国] E.E.肯明斯著 [巴西]古阿泽利绘 / 王梅译 / 人民文学出版社 / 2016
8.1 有点奇怪的故事，没有来由的、诗意的连接。不知小孩子会怎样理解。

《小熊维尼的世界》
[英国] A.A.米尔恩著 [英国] E.H.谢泼德插图 / 单益义、张文逸、薛绮玮译 /上海译文出版社 / 2021
8.7 不得不说，插图真的太厉害了，直触人心。前两册，故事，是完整且蔓延状的，自成的世界，一切事情在此间发生；后两册，诗集，是点状的离散的，但似乎更能反映童年的状态。每个人的成长中，其实都可以集采出这样的诗集，肆意，没来由，忽然的阐发，不断地重复，质问。满满的都是才本我出发的表达。一天可以做一百件事的啊，想想！

《拳击手》
[伊朗] 哈桑·穆萨维 / 山东教育出版社
8.15 想起那篇自己写过的短篇

《国王先生》
[芬兰] 席金尼 / 山东教育出版社
8.15 这个转变很神奇！

《星期五的小砂糖》
[日本] 酒井驹子 / 李丹 / 台海出版社 / 2019
8.15 油画棒感。有点神奇，有点忧郁，但不知真假。

《妙语 12 诀》
[美国] 拉里·金 / 海南出版社 / 1996
8.22 老书。简单却真切的书。通过自我奋斗而成功的成功者，热切地分享。“开放自己、热忱、愿意聆听，三点原则就可以让你成为任何一种媒体上受欢迎的健谈者。……事前准备、了解听众、浅显易懂，三点要诀就可使你成为成功的演讲人。” 找回说话的自然感。

《船在海上，马在山中》
[西班牙] 费德里科·加西亚·洛尔迦 / 戴望舒译 / 云南人民出版社 / 2020
8.25 有几首很不错

《世界 100 位作家谈写作》
上海文化出版社 / 1987.11
8.29 太有趣了，很多人当时还未离世。有几篇非常有趣的回答，想去找来作品看看。（五角丛书·第六辑）

Sep

《人类的终极问题》
袁越 / 生活·读书·新知三联书店 / 2019
9.5 太弱了。作者的整理太过啰嗦，没什么启发力。人一点也不高等，且过于自恋，瞎崇拜，浪费生命，创造力更是失传很久。

《臣服实验》
[美国] 迈克·A·辛格 / 易灵运译 / 南京大学出版社 / 2019
9.11 注定会发生的正确的事。召唤与感知。神奇的自信是神奇的无我状态。强弱争夺不复存在，不过都是复杂的背景噪音。作为一具肉身，应当奔赴的地方，就是上帝。不得不感叹不同知识背景之人类的不同，可以说是臣服，其实也可以说成不抱怨，还有天之道、一期一会。

Oct

《浮光》
吴明益 / 广西师范大学出版社 / 2015
10.1 不然呢，不然我们如何保存这些记忆呢？伯格那本书自然是影响到了他的，不过伯格那里，图片与文字不应同出于一人。每一代人的载体和偏执都如此不同，像是新的物种，那么，当下又有了什么新的记忆载体么？

《交办的技术》
[日本] 小仓广 / 林佑纯译 / 北京联合出版公司 / 后浪出版公司
10.3 以更少更集中的方式交付/交换注意力。尽可能模块化，模组化，简易化。但要注意保持灵活性，保持可增长性、可扩展性。保持合作与沟通，整合自我要求。多review，少指挥。

《数字时代的效率手册》
少数派 / 电子工业出版社 / 2021
10.5 里面大多数文章只需要给个二维码就行了。编辑核校也不够细致。少数派著，应为，少数派编著。

《麦肯锡方法》
[美国] 艾森·拉塞尔 / 张薇薇译 / 机械工业出版社 / 2020
10.6 写得比较虚，但整体逻辑是可以的。面对多样的客户，更需要压缩精力、集中精力、节约精力，所以总结出了蛮完备的方法，但不一定适合所有公司。

《即将消失的世界》
[意大利] 阿丽切·皮丘基、安德烈亚·安杰利 / 刘湃、许丹丹译 / 新星出版社 / 2018
10.6 一瞥一瞬，更像一本更精细的摄影集，不必过多联想，只需阅读，接纳很多人的不同人生，他们的当下，体验他们的现代遭遇，以及一些未被着重描述的痛苦。

Nov

《贝佐斯如何开会》
[日本] 佐藤将之 / 张含笑译 / 万卷出版公司 / 2021
11.13 真的很实用。

《设计革命：地球号太空船操作手册》
[美国] 理查德·巴克敏斯特·富勒 / 陈霜译 / 华中科技大学出版社 / 2017
11.16 老爷子真是有大胸怀，大想法，是布道者。读后觉得，需要做的事情太多，太庞杂了。

Dec

《藏在地图里的国家地理·世界》
山东省地图出版社 / 2020
12.3 啥也没讲

《置身事内 中国政府与经济发展》
兰小欢 / 上海人民出版社 / 2021
12.25 比较宏观，脉络和试图讲清的问题点，都有明确的指路。列举了经验和成功典型，但失败就不太方便着墨了，希望在提及的书里找到一些答案。算账，真的是门学问。经济体量和人口众多，任何小问题都能成为大问题，没有容许闪失的借口，这才是国有体制的基本盘。灵活性更需要以改革迭代和协同合作来达到。

《行走的柠檬：意大利的柑橘园之旅》
[英国] 海伦娜·阿特利 / 张洁译 / 商务印书馆 / 2021
12.26 谁能想到柑橘与意大利文化之间有如此多的纠葛呢！难以置信的幸福与难以置信的困苦，黑手党的隐含历史，古罗马人与阿拉伯人的灌溉遗迹。此前旅行的美好记忆，也常常与柠檬相联。
