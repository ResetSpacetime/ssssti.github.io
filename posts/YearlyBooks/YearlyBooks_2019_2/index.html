2019 阅读（下）

2020-01-01 | 阅读 | 3.1k 字 | 11 分钟

2020 阅读（上）

Jul

《啤酒有什么好喝的》
[法国] 伊丽莎白·皮埃尔、安妮·洛尔范、梅洛迪·当蒂尔克 / 吕文静 / 中信出版社 / 2017
7.6 还行，有些基础信息了解了下，比如水质对风味的影响，翻译带对照就更好了

《張岪與木心》
陳丹青 / 香港文學館 / 2019
7.23 发现温故已经出了三本，都没有买，许是将張岪与先生有关的文章辑录到了一起，变成了一本集子。巧在看到友邻转了序文，次周就到香港出差赶上书展，仅半小时时间，算是买到了。印象里只有他们二人的关系最密。他蛮少提到先生，提到了就都是些动情的话，于是我一直说去书展买情书。我也很少说到先生，甚至只认了这一位先生。先生仍在主流话语之外，甚至书中提到的BBC纪录片《文明》（目前已在线可看），第三集也把先生的段落齐齐裁去，有下载观看习惯的人，才有机会留意到他，比如黄觉。这算是什么默契吗？合适的人才可知晓他？先生也在选择吗？想来我也仅会对合适的人说起先生，也算是我们读者的默契吗？

《響～成為小說家的方法》（1-8 册）
柳本光晴 / 平川遊佐 / 尖端 / 2019
7.24 神奇的人。自我。

Aug

《职场解释系》
解释系主任 / 北京联合出版公司 / 2013
8.2 给了一套体系出来，按这个体系执行，都将是高级职场精英，有些习惯和细节都是高阶人群会注意的。但心里总莫名觉得，有结构上的缺失，不知如何表达

《闯入者》
[美国] 阿德里安·远峰 / 来兹 / 后浪丨海峡文艺出版社 / 2019
8.11 充满克制的讲述，欲放言又忽止，戛然失声，重要的是这“无声”。

《Inside Moebius》 Tome 1-6
[法国] Jean Giraud Moebius、Moebius / Stardom / 2007
8.22 6 卷本。太飘了……太能画了……或许这次不得不借助于语言，因为是 inside，要写清自己是困难的

《北海怪兽》
彭磊 / 北方妇女儿童出版社 / 2010
8.25 我们这样长大

《鱼翅与花椒》
[英国] 扶霞·邓洛普 / 何雨珈 / 上海译文出版社 / 2018
8.29 莫名地，觉得文字亲切。写入了思考和感悟，很真诚。中国文化的冲击，美食首当其冲，作者亲历过美食之本，再看当下流行的混乱、无序、铺张，实却是少了那古朴真挚的人情。

《我的造梦之路》
[日本] 今敏 / 焦阳 / 雅众文化丨新星出版社 / 2015
8.31 小鬼难缠。作品的生发。要听着音乐构思，怪不得与音乐的契合度如此高。厮磨其实是重要的。慎选合作者。

《大裂》
胡迁 / 九州出版社 / 2017
8.31 有了作品还有什么好慌乱的呢

Sep

《牛啤经:精酿啤酒终极宝典》
银海 / 中原农民出版社 / 2016
9.14 通俗易懂，很快读完，亮点不多。以及，牛啤堂真的不好喝……

《葡萄酒品鉴课堂》
[美国] 奥尔德里奇 / 孙宵祎 / 中国轻工业出版社 / 2017
9.15 没有中英对照真是要命……

《啤博士的啤酒札记》
太空精酿 / 清华大学出版社 / 2018
9.15 依次梳理常识，风格，自酿工艺。啤酒史确实又是一部世界史，足够丰满足够复杂，又有无尽的故事，对自己能喝到的每一种风格和其身后的酒厂，都开始有敬畏感。除却精确的控制，还有精确的意外，这液体中有一个复刻出的世界

《精酿啤酒革命》
[美国] 史蒂夫·欣迪 / 骆新源、沈恺、赖奕杰 / 中信出版集团 / 2017
9.15 梳理这么多事件，整理出时间线和行业的发展逻辑。验证产品，验证市场，验证商业模式，每个阶段马虎不得，牺牲者自然扑了一路。竞争与合作，市场认知，政策博弈，还有资本的贪婪。后起的市场更要足够警觉和清醒。但好消息是，性格与认知，反叛的态度，给了精酿世界源源不断的生命力

《葡萄酒有什么好喝的》
[法国] 范妮·达利耶塞克 / 司文 / 中信出版社 / 2018
9.20 还不错，信息密度还不够大，中英对照做得比较差，选酒方面实践性不强

《DK自酿啤酒入门指南》
[英国] 格雷格•休斯(Greg Hughes) / 中国轻工业出版社 / 2016
9.22 很实用的指南了，参数和数据都很细致，但是没有中英对照搞毛线，还好附录有人家分享工艺的网址……要不您放个淘宝链接

《从此开始喝精酿》
[英国] 尤安·弗格森 / 吕文静、薛赫然 / 中信出版集团 / 2017
9.20 看到了喝不到，一本四不像

《掌控习惯》
[美国] 詹姆斯·克利尔James Clear） / 迩东晨 / 北京联合出版公司 / 2019
9.26 显而易见、有吸引力、简便易行、令人愉悦。拆解“身份”概念，化解为行为。习惯即关联。原来学语言更像养成一种习惯（很像之前一本讲学英语的书）。习惯是对自我及身体的优化。缺失精确的“习惯”，即提问和以“方式”解决的能力。专注体系（方法、方式，身份出发），而非目标（方向）。反应/习惯，取决于能力。

Oct

《开罗团长+洞熊学校的三个毕业生》
[日本] 宫泽贤治 增村博 / 周龙梅 / 后浪丨湖南美术出版社 / 2019
10.18 集体工作，集体协作，本是美妙的事。发生太多被“阻断”之事，而再无法回复至原初。太多误事占满了世间，挤占视线，然后被看作应为之事。好在故事没什么具体的教益目的，是放开了空间的故事。

《消失的动物:灭绝动物的最后影像》
[英国] 埃罗尔·富勒(Errol Fuller) / 何兵 / 重庆大学出版社有限公司 / 2018
10.19 屠杀。没有意外的灭绝。人类与哺乳动物的协同谋杀。消失的中间层。害虫害鸟害兽。它们是土著。习性的消灭与悲歌。

Nov

《技术的真相》
[加拿大] 厄休拉·M. 富兰克林 / 田奥 / 南京大学出版社 / 2019
11.2 技术福祉领先于人类福祉，技术效率先于人类需求肆意发展，这就是现状。弱化人的存在，并深深陷入生产-消费的二元循环。黑客帝国是伟大的寓言。中介物，应有其有效期。

《诗的锻造:休斯写作教学手册》
[英国] 特德·休斯 / 杨铁军 / 广西人民出版社 / 2019
11.3 喜欢最后一章《词语和经验》，那一点“关键性”的流露，才能被称为“一种生命”（其实整本都在表达这一点）。mark 爱德华托马斯的那一段大海。

《培养想象》
[加拿大] 诺思罗普·弗莱 / 李雪菲 / 中国华侨出版社 / 2019
11.10 三种语言/文字的划分，完成共有的想象但保持独立，科学与文学从不同方向探索共建的世界，经典之重要，形式的整合创立与自我完成，想象给予人以无尽的滋养

《梦的化石:今敏全短篇》
[日本] 今敏 / 焦阳 / 北京联合出版公司 / 2018
11.11 一般般

《占有性个人主义的政治理论:从霍布斯到洛克》
[加拿大] C.B.麦克弗森 / 张传玺 / 浙江大学出版社 / 2018
11.16 自我权利的辨知，普及。默认互相侵犯的市场社会，与一统的权力中心（神话）。

《如何阅读一本书》
[美国] 莫提默·J. 艾德勒、查尔斯·范多伦 / 郝明义、朱衣 / 商务印书馆 / 2004
11.16 终于收尾。最后，要找到对自己有特殊价值的书。

《超越智商》
[美国] 基思·E·斯坦诺维奇(Keith E. Stanovich) / 张斌 / 机械工业出版社 / 2015
11.17 终于收尾。总之就是：不要盲目相信任何人，尤其要警惕聪明人的想法。积累正确观念的重要性，再宣扬也不为过。

《不华:柏邦妮名人访谈录》
柏邦妮 / 新世界出版社 / 2009
11.19 补完。记录了某种（当时的）状态。

《生命是什么》
[奥地利] 埃尔温•薛定谔 / 罗来欧、罗辽复 / 湖南科学技术出版社 / 2007
11.26 SENSE、MIND、FEELING，划清了物质与意识的层级与界限，实在给人极大的信心与希望！

《我的父亲母亲》
[英国] 多丽丝·莱辛 / 匡咏梅 / 南海出版公司 / 2013
11.28

《烹:烹饪如何连接自然与文明》
[美国] 迈克尔·波伦 / 胡小锐、彭月明、方慧佳 / 中信出版集团 / 2017
11.28 这就是五行吧，金木水火土田，最最基本的认识，最最低层的结构，然而我们全都遗忘了

《园丁的一年》
[捷克] 卡雷尔·恰佩克著、[捷克] 约瑟夫·恰佩克绘 / 金晨 / 青马文化丨四川文艺出版社 / 2017
11.28 春夏秋冬的生动表述

《失控:全人类的最终命运和结局》
[美国] 凯文·凯利 / 东西文库 / 新星出版社 / 2010
11.29 还好当年买来没读

《绿山墙的安妮》
[加拿大] 露西·莫德·蒙哥玛利 / 马爱农 / 浙江文艺出版社 / 2003
11.29 成长。怎么有点像小魔女亚可。

《WOOP思维心理学》
[美国] 加布里埃尔•厄廷根 / 吴国锦 / 中国友谊出版公司 / 2015
11.30 愿望，审慎地将想像内化（自发、内源性），调高期望值，然后就是干

《华尔街日报是如何讲故事的》
[美国] 威廉·E.布隆代尔 / 徐扬 / 华夏出版社 / 2006
11.30 提供了不少检查清单

Dec

《时间的形状:相对论史话》
汪洁 / 北京时代华文书局 / 2017
12.1 太口语了……

《时尚的哲学》
[挪威] 拉斯·史文德森 / 李漫 / 北京大学出版社 / 2010
12.1 很多归集整理工作，没形成什么印象。主观地觉得，大众的时尚更多是奴役，而非自主。比起“兴奋”，人群更需要“震惊”。

《一个人的诗歌史（第一部）》
刘春 / 广西师范大学出版社 / 2010
12.2 有些不必讲

《漫画之王:陈福财正传》
[新加坡] 刘敬贤(Sonny Liew) / 黄天怡 / 武汉大学出版社 / 2019
12.8 架空的角度很妙。想象力怎样强调都不为过。写出了普通人的命运，以及不可言说的记忆。质疑颇具力量，时代注脚下，看到了想象的土壤。

《远行》
[美国] 大卫·梭罗 爱默生 / 董晓娣 / 光明日报出版社 / 2012
12.8 艾默生的序最佳

《风格练习》
[法国] 雷蒙·格诺 / 袁筱一 / 人民文学出版社 / 2018
12.10 翻译很差

《为什么研究中国建筑》
梁思成 / 外语教学与研究出版社 / 2011
12.18 虽很多地方不是很懂，但感受到那种治学之严谨。批评清朝建筑的段落，很有意思。

《富兰克林自传》
[美国] 本杰明·富兰克林 / 蒲隆 / 译林出版社 / 2015
12.31 checklist 要有明确的标准，并不断自查、更正。这是一套更适合个人的方法。从事无巨细的记录和经验中，以及目睹他人的错误后，得到灵感与启发，更重要的是践行和自省。创建有限成员的共同体，极度密集的高质量交流，非常重要，且紧迫。
