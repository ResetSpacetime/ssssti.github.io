2012 阅读

2013-01-05 | 阅读 | 2k 字 | 7 分钟

Jan

《猫城记·新韩穆烈德》
老舍 / 2008 / 文汇出版社
1.13 糊涂无知且不自知，也是部分的自己，猫城记每再回味都觉精彩。

《不适之地》
[美国] 茱帕·拉希里 / 2011 / 上海译文出版社
1.21 在不适之地扎根。二代移民文学。异国生活中的思考。孤独与温暖并存。

《知无涯者》
罗伯特·卡尼格尔 / 2008 / 上海科技教育出版社
1.27 自己创造自己，用力燃烧，最后陨灭。痛心的是未能到达彼岸，牵绊总是无形，时而无从抗拒。读后对印度兴趣浓厚，诸神种姓社会欲求一探。

Feb

《知日· HI! 美术馆》
苏静 毛丹青 汤祯兆 顾铮 李清志 林竹 / 2011 / 辽宁教育出版社
2.6 nice，美术馆专题和村上的专题都不赖。后面怎样和日本人谈恋爱一篇也蛮有意思，这样的杂志真应该多出，对知识和艺术也会更好奇吧！

《我负丹青》
吴冠中 / 2004 / 人民文学出版社
2.8 “风格是作者的背影，自己看不见” 一以贯之的真诚。不过文字总不如他画作的震撼感来得强烈。

《叶赛宁抒情诗选》
叶赛宁 / 1982 / 上海译文出版社
2.17 这片土地，他爱得深沉。 http://is.gd/N1XFVl 网上有全集。摘：http://arlmy.diandian.com/post/2012-02-17/15760511

Mar

《永别了，苏珊》
[法国] 亨利·特罗亚 / 1987 / 人民文学出版社
3.10 翻译超棒！为错误的选择失去一切，正视诱惑，表象与内心，千人千面，淡然与幸福。PS这套丛书要收集来看看

《欣悦的灵魂》
切尼科夫 / 1993 / 湖南文艺出版社
3.13 读着读着心都跟着欢悦起来。对土地的爱，对人民的爱，无法停止的情怀，简单而缜密的灵魂。

《中外文学之间》
王佐良 / 1984 / 江苏人民出版社
3.16 杂选&诗人&研究，对乔伊斯和彭斯产生极大的兴趣

《活着活着就老了》
冯唐 / 2010 / 万卷出版公司
3.31 kindle 坚定了要研究些古籍的决心

Apr

None.

May

《地獄星》
伊藤润二 / 2006 / 东立出版社有限公司
邪恶的种子人人都有。众人都是孤独的。

《伊藤潤二恐怖漫畫精選》 1-10
伊藤潤二 / 1998 / 東立 8.7
1 5.27 境界！
2 5.27 境界！
4 5.28 带有扩散感的恐怖
5 5.28 孤独，却真实的双一
6 5.30 看双一，似乎能看见部分的自己
7 5.30 蛞蝓少女，霉菌
8 5.30 血玉树
9 5.31 笔友、押切怪谈
10 5.31 这本作品都好棒！

Jun

《伊藤潤二恐怖漫畫精選》 11-16
伊藤潤二 / 2000 / 東立 8.7
11 6.1 地图之城、魔音村
12 6.1 这本好纪实！坏小孩、逃兵之家、父亲的心、记忆、鬼巷都很好！
13 6.1 怪异六兄妹
14 6.3 长梦、隧道奇谭

《幽游白书》
冨樫義博 / 1990 / 河南大学出版社
恶补青春期，从富坚开始。结尾的升华，让人感动。

《笑傲江湖》
金庸 / 1994 / 生活·读书·新知三联书店
怕是第一次读完金庸的小说呢= =…玄幻神马的确实弱爆了…

《论人类不平等的起源》
卢梭 / 2009 / 上海三联书店
6.24 同情心、生命、自由……①法律和所有权的建立②官员制度的设立③法制权威向专制权威的转变

《The Art of My Neighbor Totoro》
Hayao Miyazaki / 2005 / VIZ Media LLC
6.29

Jul

《软件测试》
[美国] Ron Patton / 2006 / 机械工业出版社
7.12

《结网》
王坚 / 2010 / 人民邮电出版社
蛮受益。

《如何掌控自己的时间和生活》
[美国] 阿兰·拉金 / 2005 / 金城出版社
7.15 A-1每日做，掌握和平衡

《用户体验的要素》
Jesse James Garrett / 2007 / 机械工业出版社
7.23

Aug

《重来》
[美国] 贾森·弗里德 [丹麦] 戴维·海涅迈尔·汉森 / 2010 / 中信出版社
8.6

《宇宙尽头的餐馆》
[英国] 道格拉斯·亚当斯 / 2012 / 上海译文出版社
8.12

《白》
原研哉 / 2012 / 广西师范大学出版社
8.14 颜色的感性力量

Sep

《不二》
馮唐 / 2011 / 天地圖書有限公司
9.7 kindle 在公司看不二是件刺激的事儿

《笋瓜自传》
帕里斯 / 2005 / 人民文学出版社
9.13 满满的都是爱

《抬高房梁，木匠们；西摩：小传》
[美国] J.D.塞林格 / 2009 / 人民文学出版社
9.16 完美与不完美，生活之美与细节之美。

《好色的哈姆莱特》
小白 / 2009 / 人民文学出版社
9.19 几篇蛮好

《出版人》
[英国] 汤姆·麦奇勒 / 2008 / 人民文学出版社
9.22 趣闻轶事，只是翻译得太小学生了….

《东榔头》
杨葵 / 2012 / 湖南文艺出版社
9.23 soso

《西棒槌》
杨葵 / 2012 / 湖南文艺出版社
9.24 soso

Oct

《汉品》 01
左靖 / 2012 / 金城出版社
10.21 重返传统，这种坚持很让人佩服。康熙字典体，美绝。

《十九札》
朱青生 / 2011 / 世界图书出版公司 / 后浪出版公司
10.24 唯有说相见恨晚，未来得及成为专家就四处发表见解的自己，真是十分可笑

《网民的狂欢》
[美国] 安德鲁·基恩 / 2010 / 南海出版公司
10.25 更大程度上是记述和警示，而非论证

《信任代理》
[美国] 克里斯·布罗根 / 2011 / 万卷出版公司
10.27 个人品牌的塑造，信任、真诚、真实是社会化时代的根基。

Nov

《认知盈余》
[美国] 克莱·舍基 / 2011 / 中国人民大学出版社
11.1 以一个议题为中心进行的周边思考非常丰满。动机、方式、机会，创造社会价值。

《创意的秘密》
史蒂夫•哈里森 / 2012 / 世界图书出版公司 / 后浪出版公司
11.7 “问题/解决机理”+“恰切截断”，打扰式营销尚未没落。翻译略差。Mark《戈西奇之书》

《道连·葛雷的画像》

《游戏改变世界》
[美国] 简·麦戈尼格尔 / 2012 / 浙江人民出版社
11.20 不如说，如何巧妙地利用和引导大众游戏生活、探索未知、感知外界，在亲历中成长和学习，因此这种研究和设计的能力却是更为重要。

《快速阅读》
东尼·博赞 / 2009 / 中信出版社
11.25 要是有本练习册，就更好了

12.4 够干。不光是卖家的水平，买家的水平也能有提升。

《兰波作品全集》
兰波 / 2011 / 作家出版社

《牡蛎男孩忧郁之死》
